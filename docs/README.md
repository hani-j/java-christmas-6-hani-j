## 🎄크리스마스 프로모션🎁

## 기능 목록

한줄 핵심 기능 :

1. 날짜가 평일, 주말, 특별일 인지 비교 한다.
2. 날짜별 할인 혜택을 구한다.
3. 메뉴 저장, 비교, 카테고리, 가격
4. 주문목록

- [ ] 방문 날짜 검사 기능

- [ ] 메뉴와 개수 검사 기능
    - [x] [ERROR] 없는 메뉴인 경우
    - [x] [ERROR] 개수가 1이상이 아닌 경우
    - [x] [ERROR] 주문 형식이 다른 경우

    - [x] [ERROR] 개수가 20개 넘는 경우
    - [x] [ERROR] 음료만 주문한 경우

- [x] day type 반환 기능
